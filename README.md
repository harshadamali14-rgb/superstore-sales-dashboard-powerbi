# superstore-sales-dashboard-powerbi

This project showcases an interactive Power BI dashboard built using the Sample Superstore dataset. It provides comprehensive insights into sales performance, profit, orders, returns, customer segments, product categories, and regional trends. The dashboard enables users to monitor key business metrics and make data-driven decisions through interactive visualizations and filters.

Key Features
📊 Sales, Profit, and Orders KPI Cards
📈 Monthly Sales & Profit Trend Analysis
🌍 Region, State, and City Performance Analysis
🛍️ Category and Sub-Category Sales Analysis
👥 Customer Segment Analysis
📦 Returns Analysis
🎯 Interactive Slicers and Drill-Down Reports
📌 Data Cleaning, Modeling, and DAX Calculations
Tools & Technologies
Power BI
Power Query
DAX
